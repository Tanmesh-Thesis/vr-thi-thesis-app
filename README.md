# Locispace (place to create mind palace)

## Virtual Reality based application to visualise and memorize concepts

#locispace is a virtual learning environment where you build a 3D mind palace using Loci technique in order to remember complex concepts and check your progress with learning games.

To know more about the project visit the WIKI page of this repository
